# Security

This document outlines the security assumptions of CavePay Wallet Core.

---

## Model

- Non-custodial
- Local-only key management
- No server-side signing

---

## In Scope

- Malicious dApps
- Untrusted RPC endpoints
- UI-level signing abuse

---

## Out of Scope

- Compromised operating systems
- Browser or extension malware
- Physical device access

---

## User Responsibility

- Secure seed backups
- Device security
- Transaction verification

---

## Disclosure

Security issues should be reported privately.
